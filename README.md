Stylinx is a modern React Native E-Commerce mobile application built with Redux state management and Firebase authentication. The app allows users to browse products, view details, add items to cart, place orders.

Tech stack:
React Native
Redux Toolkit
React Navigation
Firebase Authentication
JavaScript
REST API (EscuelaJS API)

<!-- To install packages -->
npm install 

<!-- To run application -->
 npm run android
